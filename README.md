# ExperimentalFeatures
(しばらくの間PocketMine-MPには実装されないであろう)新しい要素を追加することを目的としたプラグイン。  
ワールドやプレイヤーデータが破損するため、必ずバックアップを取ってから使用してください。  
PocketMine-MP本体を変更しないのが原則なので(悪い意味で)素晴らしい実装がたくさん見られます。
